# 第01章 云开见月

夏末的阳光，透过树叶的缝隙，洒落满地金黄。校门口熙熙攘攘，涌动的人潮里，交织着家长期许的目光，也满含着少年人对未来的向往。

穿过注册处拥挤的人群，牧云雨正准备朝教学楼走去，忽然，一阵急促的脚步声自身后传来，伴随着一声略带慌乱的“啊！小心”。牧云雨下意识回头，还未来得及看清，就被一个身影结结实实地撞了个趔趄。

“对不起！对不起！”女孩手忙脚乱地站稳，怀里抱着的几本书散落一地。

牧云雨揉了揉被撞疼的肩膀，低头看向肇事者。女孩扎着简单的马尾，几缕碎发因为奔跑而贴在额前，白皙的脸颊因急促的呼吸而泛着红晕。她正蹲在地上，着急地捡拾散落的书本。

牧云雨微微一怔，“开学第一天就这么拼命？”他蹲下身，帮她一起整理着散落的书本。

“谢谢……”女孩他手中接过书本，抱在怀里，眼神里带着歉意和感激，“我赶着去看分班，没注意到你在前面，没撞疼你吧？”

“没事。”牧云雨淡淡回应，目光却在她脸上多停留了一秒。不知为何，他觉得这个冒失的女生有点眼熟，却想不起在哪里见过。

“那我先走了！”女孩朝他挥挥手，又匆匆向公告栏方向跑去。

牧云雨望着她远去的背影，轻轻摇头。他向来不喜欢毛毛躁躁的人，尤其是开学第一天就给他来个“意外惊喜”的这种。

走到分班公告栏前，那里已经围了不少学生。牧云雨踮起脚尖，在密密麻麻的名字中寻找自己的班级。

“高一（3）班...”他喃喃自语，目光继续在名单上移动。

“又是你？”旁边传来一个清脆的声音。

牧云雨转头，看见女孩正站在他身侧，仰头看着公告栏，“我在（3）班，你呢？”

“我也在（3）班”

“看来我们挺有缘啊，撞人撞出个同班同学。”她笑道，眼睛弯成了月牙。

“这种缘分还是少点为好。”牧云雨面无表情地说。

苏挽月不以为意，反而伸出手：“正式认识一下，我是苏挽月。”

牧云雨犹豫了一瞬，还是握了上去：“牧云雨。”

“牧云雨...”苏挽月重复了一遍，“这名字真好听，像小说男主角。”

牧云雨没有接话，只是默默收回手。他向来不擅长应付这种自来熟的人。

两人按照指示牌找到教学楼三层的教室。教室里已经坐了不少同学，三三两两地交谈着，空气中弥漫着新生见面特有的好奇与拘谨。

牧云雨选了个靠窗的位置坐下，苏挽月则在他斜后方落座。

“嘿，牧云雨。”苏挽月轻轻拍了拍他的肩膀，“借支笔可以吗？我的不知道丢在哪里了。”

牧云雨叹了口气，从笔袋里拿出一支黑色水笔递过去。

“谢谢！我保证这次不会弄丢。”苏挽月信誓旦旦。

一位戴着眼镜的年轻女老师很快走进教室，“我是你们的班主任梁紫萱。”她简单自我介绍后，开始讲解校规校纪和课程安排。牧云雨专注地听着，偶尔在笔记本上记下重点。

“接下来，我们选一下临时班委。”班主任说，“有谁愿意主动承担吗？”

教室里一片安静，新生们大多低着头，不愿第一个举手。

“老师，我推荐牧云雨同学。”一个清脆的声音突然响起。

牧云雨猛地回头，看见苏挽月正举着手，脸上带着狡黠的笑容。

“为什么推荐他？”班主任好奇地问。

“因为他看起来就很可靠。”苏挽月说得理所当然。

几个同学低声笑起来。他瞪了苏挽月一眼，她却回以一个无辜的表情。

最终，在无人反对的情况下，牧云雨莫名其妙地成了临时班长。

休息时，牧云雨走到苏挽月桌前：“你什么意思？”

“帮你争取表现机会啊。”苏挽月理直气壮，“不用谢我。”

“我看起来像是需要这种机会的人吗？”牧云雨皱眉。

苏挽月歪头打量他：“你看起来太独来独往了，当班干部有助于你融入集体。”

牧云雨一时语塞，他从没见过如此自作主张的人。

“再说了，”苏挽月从书包里掏出那支黑色水笔，轻轻放在桌上，“作为借笔的回报，我总得为你做点什么。”

牧云雨拿起笔，无奈地摇头：“我们扯平了。”

“那可不行。”苏挽月笑道，“我还欠你一个人情呢，毕竟早上撞了你。这样吧，午饭我请你，食堂应该已经开了。”

牧云雨本想拒绝，但看着苏挽月期待的眼神，话到嘴边又咽了回去。

“随便。”他最终吐出两个字。

午餐时间，苏挽月果然兑现承诺，请牧云雨吃了食堂的招牌牛肉面。两人面对面坐着，气氛有些微妙地安静。

“其实，我早就认识你。”苏挽月突然说。

牧云雨抬头：“什么？”

“区里的书法作品展，十三中的优秀作品《行云流水》，作者就是你吧？”苏挽月眼中闪着光，“我特别喜欢那幅作品，所以在公告栏看到你名字时，我就认出来了。”

牧云雨有些惊讶。他确实参加过那场比赛，但没想到会有人记得那么清楚。

“所以早上你是故意撞我的？”他半开玩笑地问。

“当然不是！”苏挽月急忙否认，随即又笑了，“不过，能认识你，我很高兴。”

牧云雨低头吃面，没有回应，面汤的热气氤氲上来，没让人看见他眼底一闪而过的笑意。

下午的入学教育结束后，学生们陆续离开教室。牧云雨整理好书包，发现苏挽月站在门口等他。

“一起走吧，班长大人。”她笑着说。

夕阳西下，两人的影子在走廊上拉长。牧云雨看着身旁哼着歌的苏挽月，突然觉得，这个开学日虽然充满意外，但似乎并不坏。

“喂，牧云雨。”走到校门口时，苏挽月叫住他，“明天见。”

她挥手告别，转身汇入人流。牧云雨站在原地，直到那个扎着马尾的背影消失在视线中。

微不足道的相遇，微不足道的开端。但谁又能预料，这些微不足道的瞬间，将会编织成怎样的故事？

牧云雨抬头，天空中云雨初散，一弯新月悄然挂上天际。
