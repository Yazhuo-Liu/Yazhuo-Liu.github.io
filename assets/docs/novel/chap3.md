# 第03章 雨季的暗涌

周五的放学铃声总是比往常更悦耳些，像是一种解脱的信号。喧闹声潮水般退去，高一（3）班的教室里只剩下负责值日的牧云雨和苏挽月。

窗外的天色有些阴沉，云层压得很低，闷热的空气预示着一场暴雨即将来临。

牧云雨一言不发地挥动着扫帚，动作利落且有条理。相比之下，苏挽月显得有些心不在焉，她一边擦黑板，一边时不时透过窗户看向操场方向------那里，许予微和杜诗婷正并肩走过。

"哎，班长。"苏挽月突然停下手中的动作，趴在讲台上，下巴抵着黑板擦，眼神晶亮，"你觉不觉得，予微和诗婷特别般配？"

牧云雨停下手中的动作，顺着她的目光看了一眼窗外。操场上，杜诗婷正仰头跟许予微说着什么，笑容灿烂，而许予微依旧是那副温和的模样，手里提着杜诗婷的书包。

"哪里般配？"牧云雨收回视线，语气平淡。

"青梅竹马啊！而且你看，许予微多宠她，书包都帮她拿。"苏挽月一副磕到了的表情，语气里满是兴奋，"诗婷可是跟我说了，她觉得许予微对她肯定有意思，就是那层窗户纸还没捅破。作为好闺蜜，我得帮帮他们。"

牧云雨看着苏挽月那副兴致勃勃的样子，眉头微微蹙起。作为旁观者，他那双善于观察的眼睛看到的画面却截然不同
。

许予微虽然提着书包，但身体却微妙地向外倾斜，与杜诗婷保持着一种礼貌而疏离的距离；杜诗婷笑得开心时想去拉他的袖子，许予微却"恰好"抬手看表，不动声色地避开了
。

"你最好别多管闲事。"牧云雨冷冷地提醒，"有些'般配'可能只是你的一厢情愿。"

苏挽月不满地鼓起脸颊："你这人怎么总是这么泼冷水？上次语文课也是，这次也是。浪漫一点会死吗？"

"盲目的浪漫通常是悲剧的开端。"牧云雨将垃圾倒入垃圾桶，发出"哗啦"一声响，"快点干活，要下雨了。"

话音刚落，一道闪电划破长空，紧接着是震耳欲聋的雷声。豆大的雨点噼里啪啦地砸在窗玻璃上，瞬间将世界笼罩在一片雨幕中。

两人匆匆做完最后的收尾工作，跑到教学楼一楼大厅时，雨势已经大得惊人。门口聚集了不少没带伞的学生，正焦急地望着天。

人群中，牧云雨一眼就看到了许予微和杜诗婷。杜诗婷正懊恼地翻着空空如也的书包，而许予微手里拿着一把深蓝色的折叠伞。

"机会来了！"苏挽月眼睛一亮，根本来不及听牧云雨的阻拦，就以此生最快的速度冲了过去，一把拍在许予微的肩膀上。

"许学委！这么巧啊！"苏挽月笑嘻嘻地打招呼，然后迅速将一脸懵懂的杜诗婷推到许予微面前，"诗婷没带伞，这么大的雨，你肯定不忍心让她淋雨或者等家长吧？正好你们顺路，你就发扬一下绅士风度呗！"

这套动作行云流水，显然是她在脑海里预演过的"神助攻"。

周围有几个同学投来起哄的目光
，杜诗婷羞红了脸，低下头捏着衣角，满心期待地等着许予微的回应。

然而，空气却突然安静了几秒。

许予微脸上的温和笑容在那一瞬间似乎僵硬了一瞬。他看了一眼低着头的杜诗婷，又看了一眼满脸期待望着他的苏挽月，握着伞柄的手指微微收紧，指节泛白。

"挽月，"许予微的声音在嘈杂的雨声中显得有些沉，"那你呢？"

"啊？我？"苏挽月愣了一下，"我没事啊，我稍微等雨小一点......"

"这雨一时半会儿停不了。"许予微打断了她，上前一步，将手中的伞递到了苏挽月面前，目光直直地锁住她的眼睛，语气里带着一种从未有过的强硬，"这把伞给你。我和诗婷在这一起等家里人来接，我有外套，可以借给她挡风。"

这一幕完全出乎了所有人的预料。

杜诗婷猛地抬起头，错愕地看着许予微，眼里的光瞬间黯淡下去。周围起哄的声音也戛然而止，气氛变得极其尴尬。

苏挽月看着递到眼前的伞，整个人都懵了。这剧本不对啊！许予微不应该顺势搂着杜诗婷走进雨里上演偶像剧吗？把伞给自己算怎么回事？

"不不不，我怎么能拿你的伞......"苏挽月手忙脚乱地推拒。

"拿着。"许予微坚持道，眼神里甚至带了一丝隐忍的执拗。

就在这僵持不下的尴尬时刻，一只骨节分明的手突然从侧后方伸出，一把扣住了苏挽月的手腕，将她往后拉了一步，远离了那把深蓝色的伞。

众人回头，看见牧云雨面无表情地站在那里，另一只手里撑开了一把纯黑色的长柄伞。

"不用麻烦学习委员了。"牧云雨的声音清冷，穿透雨幕，"她跟我顺路，我送她。"

苏挽月惊讶地回头："啊？可是你家不是......"

牧云雨没给她说话的机会，淡淡地扫了许予微一眼。那眼神平静却犀利，仿佛看穿了许予微温和面具下所有的隐秘心思
。许予微的眼神微微一闪，没有退让，两人在空气中无声地对峙了一秒。

"走了。"

牧云雨手上微微用力，带着还没反应过来的苏挽月走进了雨中。黑色的伞面很大，却明显向苏挽月那边倾斜了大半，牧云雨的左肩瞬间被雨水打湿。

留在原地的许予微看着两人离去的背影，慢慢垂下了拿着伞的手。

"予微哥......"杜诗婷小心翼翼地拉了拉他的衣袖，声音带着哭腔，"你是不是......不喜欢我这么麻烦你？"

许予微回过神，转头看向杜诗婷。那一刻，他又变回了那个温和完美的邻家哥哥。他脱下校服外套披在杜诗婷身上，叹了口气，语气温柔却残忍："傻瓜，你是妹妹啊，哥哥照顾妹妹怎么会嫌麻烦？只是伞只有一把，怕你淋湿而已。"

雨幕中。

苏挽月被牧云雨拽着走了一段路才回过神来，她挣扎了一下："牧云雨，你慢点！还有，谁跟你顺路了？"

牧云雨停下脚步，低头看着她。

"我要是不把你拉走，你打算在那尴尬站到什么时候？"

苏挽月语塞，想起刚才杜诗婷受伤的眼神和许予微奇怪的反应，她有些抓狂地挠了挠头："刚才到底怎么回事啊？予微他是不是脑子短路了？放着喜欢的青梅竹马不管，非要给我伞？"

牧云雨看着她那一脸茫然的样子，不知该气还是该笑。

"苏挽月。"

"干嘛？"

"你以后，"牧云雨抬手，轻轻弹了一下她的额头，"少自作聪明。有些事情，全世界都看明白了，就你是个傻子。"

"喂！你会不会好好说话！"苏挽月捂着额头抗议。

牧云雨没理会她的抗议，重新迈开步子："去前面的公交站，我看着你上车。"

雨还在下，黑色的伞下，两人的距离被迫缩得很短。苏挽月闻到了牧云雨身上淡淡的墨水味，混合着雨水的潮湿气息。她偷偷瞄了一眼牧云雨被打湿的半边肩膀，心里那种奇怪的烦躁感莫名消散了一些。

"牧云雨。"

"又怎么了？"

"虽然你这个人嘴毒又面瘫，但刚才......谢了啊。"

牧云雨看着前方迷蒙的雨雾，嘴角微不可察地扬起一个极浅的弧度。

"一顿牛肉面，扯平了。"
