# 第04章 裂痕与流言

周一的早晨，雨过天晴，空气里混杂着泥土和青草的味道。但高一（3）班的气氛，却比那个暴雨的傍晚还要闷热潮湿。

苏挽月走进教室时，手里提着两份热腾腾的豆浆和饭团。这是她特意早起去校门口排队买的------杜诗婷最喜欢这家的糯米饭团。她想着，周五那场尴尬的"助攻"可能让杜诗婷心情不好，作为闺蜜，必须得拿出点诚意来赔罪。

"诗婷，早啊！"苏挽月扬起笑脸，走到杜诗婷座位旁，将早餐放在她桌上，"给你买了最爱的肉松饭团，还是热的呢。"

杜诗婷正在背英语单词，闻言抬起头。她的眼下有一层淡淡的乌青，似乎周末没睡好。她看了一眼那份早餐，并没有像往常那样欢呼着接过，而是甚至没有伸手去碰。

"不用了，我在家吃过了。"杜诗婷的声音很轻，却带着一种拒人于千里之外的冷硬。

"啊？你平时不是说家里的早餐吃不饱吗？"苏挽月有些发愣，手尴尬地停在半空，"你就当零食吃呗，我排了好久的队......"

"我说吃过了。"杜诗婷突然提高了音量，手中的英语书重重地合上，"苏挽月，你能不能不要总是自以为是地安排别人的事情？"

教室里的嘈杂声瞬间像被按了暂停键。前排的几个女生回过头，眼神在两人之间探究地打转。

苏挽月彻底僵住了。她从未见过杜诗婷用这种厌烦的语气跟她说话。在她的记忆里，杜诗婷总是那个跟在她身后，笑嘻嘻地挽着她胳膊的小妹妹。

"哟，这是怎么了？"

一个不合时宜的声音打破了僵局。那是坐在后排的一个男生，平时最爱传八卦，"听说周五放学，咱们许大才子为了苏挽月，把青梅竹马都晾在一边了？杜诗婷，你这是吃醋呢，还是迁怒啊？"

"你闭嘴！"杜诗婷猛地站起来，眼圈瞬间红了，狠狠瞪了那个男生一眼，转身冲出了教室。

"哎，诗婷！"苏挽月下意识想追，却被一只手拦住了去路。

牧云雨不知何时站在了过道上，手里拿着一叠作业本，面无表情地挡在她面前。

"交作业。"他冷冷地说。

"可是诗婷她......"苏挽月急得跺脚。

"她现在最不想见的人就是你。"牧云雨看着她焦急的眼睛，语气虽然淡漠，却透着一丝不容置疑的冷静，"你追上去，除了让她更难堪，没有任何作用。"

苏挽月愣了一下，看着牧云雨深沉的眸子，那种想要冲出去的冲动莫名被压了下来。她颓然地坐回座位，看着桌上渐渐变凉的饭团，心里像是堵了一团湿棉花。

"为什么啊......"她喃喃自语，"我明明是想帮她。"

"有些忙，别人没开口，就别瞎帮。"牧云雨将作业本放在讲台上，路过她身边时，用只有两个人能听到的声音说，"先把你自己的一团乱麻理清楚吧。"

第一节
课间，许予微才姗姗来迟。他似乎也听到了些风言风语，脸色不太好看。走进教室时，他径直走到苏挽月桌前。

"挽月，周五那天......你没事吧？有没有淋湿？"他的声音依旧温和，但眼神里却藏着一丝急切。

苏挽月正趴在桌子上郁闷，闻言抬起头，却下意识地看向许予微身后的空座位------杜诗婷还没回来。

"我没事，牧云雨借我伞了。"苏挽月避开了许予微关切的目光，有些生硬地说，"予微，你应该去看看诗婷，她刚才......情绪不太好。"

许予微的动作微微一顿，眉头轻皱："她在闹脾气，等她冷静下来就好了。倒是你，我听说有人乱传闲话，你别放在心上。"

"我在意！"苏挽月突然有些烦躁，她看着许予微那副理所当然的样子，第一次觉得这个温润如玉的男生有些陌生，"许予微，大家都说你和诗婷是一对，我也一直以为你对她......如果你不喜欢她，能不能别给她那些让人误会的特权？能不能别总让她帮你拿书、别总跟她一起回家？"

许予微愣住了，他显然没想到一向大大咧咧的苏挽月会说出这番话。

"我只是把她当妹妹......"

"那就像个哥哥一样对她，而不是像个备胎一样吊着她！"

苏挽月说完这句话，自己都吓了一跳。她猛地站起身，不想再面对许予微错愕的表情，转身向教室外走去。

走廊的尽头是学校的宣传栏。苏挽月漫无目的地走到那里，深吸了一口外面的空气，试图平复胸口的憋闷。

"骂得挺精彩。"

苏挽月回头，看见牧云雨正靠在宣传栏旁边的柱子上，手里依然拿着那支无论何时都在转的笔。他似乎在那里站了有一会儿了。

"你偷听我们说话？"苏挽月没好气地瞪了他一眼。

"路过。"牧云雨淡淡地回应，目光投向宣传栏上的一张新海报，"而且你们声音那么大，想不听见都难。"

苏挽月顺着他的目光看去，那是一张\*\*"南都一中书法社招新"\*\*的海报。海报正中央，印着一幅笔力遒劲的字，虽然没有落款，但那熟悉的风格让苏挽月心头一跳。

"这是你写的？"她指着海报问。

"以前的练习稿，被社长拿去用了。"牧云雨承认得很大方。

苏挽月看着那行云流水的字迹，心情不知为何平静了许多。她转头看向牧云雨："喂，班长。你说我是不是真的很笨？我想撮合他们，结果搞得大家都尴尬；我想骂醒许予微，结果好像也没什么用。"

牧云雨停下转笔的动作，侧过头看着她。阳光透过走廊的窗户洒在他身上，给他清冷的轮廓镀上了一层暖色。

"你不笨，你只是太想当然。"牧云雨的声音难得地柔和了几分，"许予微享受被杜诗婷依赖的感觉，但他追逐的是那天边不可得的月亮。杜诗婷知道自己是'妹妹'，但她在这个身份里越陷越深，不愿意醒来。"

苏挽月听得似懂非懂："什么月亮？"

牧云雨看着她那双清澈却迷糊的眼睛，无奈地摇了摇头，嘴角勾起一抹若有似无的笑意。

"没什么。"他站直身体，"书法社在招新，你要不要去看看？练练字，能让人心静，少闯点祸。"

苏挽月撇撇嘴："嫌弃我就直说。不过......"她眼睛转了转，"如果我去报名，看在班长大人的面子上，能不能免面试？"

"不能。"牧云雨拒绝得干脆利落，转身往教室走，"公私分明。"

"切，小气鬼！"苏挽月在他身后做了个鬼脸，但脚步却轻快地跟了上去。

两人一前一后走进教室，这一幕正好落在刚回来的杜诗婷眼中。她坐在座位上，红肿的眼睛死死盯着苏挽月脸上那还没来得及收回的笑容，手指紧紧抓住了衣角。

她看见了许予微对苏挽月的偏爱，也看见了那个总是冷冰冰的牧云雨对苏挽月的特殊。

凭什么？

凭什么苏挽月什么都不做，就能得到所有人的目光？而自己小心翼翼维护了这么多年的感情，却被她一句"乱点鸳鸯谱"搞得像个笑话？

杜诗婷低下头，从课桌里拿出一张刚刚在楼下拿到的书法社报名表。

那是她原本打算拿给许予微看的，因为许予微说过喜欢字写得好的女生。但现在......

她拿起笔，在报名表上重重地写下了自己的名字。力透纸背，像是在宣战。

窗外，风又起了，吹动着树叶沙沙作响。裂痕已经产生，而流言，才刚刚开始它的表演。
