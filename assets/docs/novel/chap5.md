# 第05章 笔锋下的暗战

书法社的活动室设在艺术楼顶层，窗外便是那个有着人字碑的小花园。周三下午社团招新面试，空气里弥漫着淡淡的松烟墨香。

牧云雨坐在长桌主位，身穿整洁的校服衬衫，袖口挽到手肘处，露出一截线条流畅的小臂。他正在研磨，神情专注而冷淡，仿佛周围嘈杂的人声都与他无关。

"下一个。"牧云雨头也不抬地喊道。

门被推开，走进来的却是两个人。

走在前面的是杜诗婷，她今天特意把马尾扎得很高，显得干练而紧绷。紧跟在她身后的，是一脸温和笑容的许予微。

教室里顿时响起一阵窃窃私语。 "那是许予微吧？他也来面试书法社？"
"大概是陪杜诗婷来的吧，你看他们形影不离的。"

坐在角落里候场的苏挽月，听到这些议论，下意识地捏紧了手中的报名表。她原本想和杜诗婷打招呼，但杜诗婷目不斜视地从她身边走过，仿佛她是空气。

"我也报名。"许予微走到长桌前，将两份报名表放在牧云雨面前，那是他和杜诗婷的，"不介意吧，牧社长？"

其实牧云雨还不是社长，只是现任社长高三学业繁忙，将这次招新全权交给了他这个"接班人"。

牧云雨手中的墨锭微微一顿，抬眼扫过许予微，又看了看旁边抿着嘴唇、眼神倔强的杜诗婷。最后，他的目光落在了角落里的苏挽月身上。

"苏挽月，你也过来。"他淡淡开口。

苏挽月愣了一下，在众人的注视下磨磨蹭蹭地走了过去，站在了杜诗婷的另一侧。

原本宽敞的长桌前，瞬间形成了诡异的"三足鼎立"。

"既然都来了，那就一起试笔吧。"牧云雨从笔架上取下三支毛笔，依次摆开，"不考别的，就写一个字------'心'。"

"心？"许予微挑了挑眉，"这题目有点意思。"

杜诗婷二话不说，率先拿起笔。她深吸一口气，蘸饱了墨，手腕悬空，起笔极重。她学过几年楷书，基础扎实，一笔一划都力求完美。横平竖直，结构严谨，一个端端正正的颜体"心"字跃然纸上。

写完后，她有些挑衅地看了一眼苏挽月。

许予微在一旁看着，赞许地点头："诗婷的字还是这么规矩，功底很深。"

这句夸奖让杜诗婷的脸色稍稍缓和，她期待地看向牧云雨。

牧云雨只是扫了一眼，面无表情："结构工整，但法度太森严。心若被框得太死，就只是个牢笼。"

杜诗婷的笑容瞬间僵在脸上，握笔的手指猛地收紧。

"轮到我了。"许予微笑着打圆场，提起笔。他的字是瘦金体风格，笔锋凌厉，透着一股书卷气和不易察觉的傲气。

牧云雨看后，给出了评价："锋芒太露，有些喧宾夺主。字是好字，但不够静。"

许予微不置可否地耸耸肩，目光却越过牧云雨，落在他身旁的苏挽月身上："挽月，到你了。别紧张，随便写写。"

苏挽月看着面前洁白的宣纸，手心微微出汗。她没系统学过书法，唯一的经验就是小时候被爷爷抓着练过几天大字。

她深吸一口气，闭上眼。脑海里莫名浮现出那天暴雨中，牧云雨倾斜的伞，还有他那句"露水见到阳光就会消散"
。

心。什么是心？
是那天的雨，是今天的风，是想要抓住点什么，却又不得不放手的悸动。

她睁开眼，提笔落下。
不同于杜诗婷的严谨，也不同于许予微的凌厉。她的动作有些生涩，但极其流畅，甚至带着几分潦草。墨迹晕染，那一笔"卧钩"拉得很长，像是一弯随意的月亮，透着一股不受拘束的灵动。

写完，她不好意思地吐了吐舌头："献丑了，好像有点歪。"

许予微正要开口安慰，却听见牧云雨的声音响起。

"通过。"

这两个字掷地有声。

"什么？"杜诗婷不可置信地瞪大了眼睛，"她连笔锋都藏不住，结构也散了，为什么是通过？"

"书法不仅仅是技法，更是心境。"牧云雨拿起苏挽月的那张纸，指着那有些稚嫩的笔触，"杜诗婷，你的'心'字太满，每一笔都想争，每一笔都怕输，最后反而困住了自己。而她的字......"

牧云雨顿了顿，目光深深地看了一眼苏挽月，声音低沉了几分："虽然技法生疏，但意态舒展，落笔无悔。这才是'心'该有的样子。"

"因为她是苏挽月，所以她做什么都是对的，是吗？"杜诗婷的声音在颤抖，眼泪在眼眶里打转。

"诗婷！"许予微皱眉，想拉住她。

杜诗婷一把甩开许予微的手，直视着牧云雨："牧云雨，你敢说你没有私心吗？"

整个活动室死一般的寂静。

牧云雨迎着她质问的目光，既没有慌乱，也没有解释。"在书法社，我只看字。你如果觉得不公，可以去向校团委投诉。"

"好......很好。"杜诗婷咬着牙，眼泪终于掉了下来。她一把抓起自己那张写得端端正正的字，揉成一团，狠狠砸进废纸篓，"我不稀罕进这种社团！"

说完，她转身冲出了大门。

"诗婷！"许予微脸色一变，看了一眼苏挽月，又看了一眼跑出去的杜诗婷，最终还是叹了口气，"挽月，牧社长，抱歉。"说完，他转身追了出去。

一场面试，不欢而散。

原本热闹的活动室此时只剩下寥寥几人。苏挽月站在原地，看着许予微追出去的背影，心里五味杂陈。

"还愣着干什么？"牧云雨的声音在她耳边响起。

苏挽月回过神，看见牧云雨正拿着一块墨锭，递到她面前，"既然通过了，就要干活。把这方砚台洗干净，这周的任务。"

苏挽月接过沉甸甸的墨锭，有些委屈："牧云雨，我是不是又搞砸了？其实诗婷的字真的比我好......"

"好坏不是只看表面的规整。"牧云雨坐回椅子上，重新拿起笔，低头在一张新的宣纸上写着什么，"而且，把她留下，只会让她更痛苦。在这里，她赢不了你，更赢不了自己在意的那个人。"

苏挽月似懂非懂："你在说许予微吗？可是许予微刚才一直在夸她啊。"

"有时候，哪怕是善意的谎言，也是一种残忍。"牧云雨写完最后一笔，放下笔，"去洗砚台吧。"

苏挽月哦了一声，转身走向水池。

等她的背影消失在门后，牧云雨才轻轻拿起刚才写好的那张纸。
纸上并不是什么名言警句，而是一个临摹的字。
正是苏挽月刚才写的那个有些歪斜、却灵动异常的"心"。

他看着那个字，手指轻轻摩挲过未干的墨迹，嘴角勾起一抹无奈又纵容的弧度。

"明明写得像狗爬一样......"他低声自语，"可偏偏，越看越顺眼。"

窗外，风吹过树梢，带走了夏末最后的一丝燥热。少年隐秘的心事，就像这墨迹一样，晕染在纸上，虽然无声，却力透纸背。
